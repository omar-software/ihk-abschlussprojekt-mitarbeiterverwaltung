# IHK Abschlussprojekt – Mitarbeiterverwaltungssystem

Dieses Repository beschreibt mein IHK-Abschlussprojekt im Bereich Fachinformatiker für Anwendungsentwicklung.

## Projektbeschreibung

Im Rahmen meines Abschlussprojekts habe ich eine webbasierte Anwendung zur effizienten Verwaltung von Mitarbeiterdaten, Arbeitszeiten, Dokumenten und Schichtplänen konzipiert und entwickelt.

## Funktionen

* Mitarbeiterverwaltung
* Rollenbasierte Authentifizierung und Autorisierung
* Zeiterfassung
* Dokumentenmanagement
* Schichtplanverwaltung
* Benachrichtigungssystem
* Admin-Dashboard

## Technologien

* C#
* ASP.NET Blazor Server
* Microsoft SQL Server
* Dapper
* BCrypt
* Bootstrap

## Technische Schwerpunkte

* Service-Architektur
* Rollenbasierte Navigation
* Sichere Passwortverarbeitung
* Datenvalidierung mit DataAnnotations
* SQL-basierter Datenzugriff mit Dapper

## Hinweis

Der ursprüngliche Quellcode wurde im Unternehmenskontext entwickelt und kann aus Vertraulichkeits- und Urheberrechtsgründen nicht öffentlich veröffentlicht werden.

Dieses Repository dient ausschließlich zur Beschreibung der fachlichen und technischen Inhalte des Projekts.
